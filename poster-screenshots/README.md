# Poster Screenshots

Screenshots for the UniGuard poster. The poster displays them automatically.

## Automated capture

1. Start the app: `npm run dev` (or `dev:all`)
2. Install Playwright: `npx playwright install chromium`
3. Run: `npm run poster:screenshots`
4. A browser will open — log in if prompted
5. Wait ~15 seconds; the script will capture screenshots

## Manual

Add PNG/JPG files with these names:
- `dashboard.png` — Wallet Hub / Dashboard
- `nexus.png` — Financial Nexus (3D view)
- `receipt.png` — Receipt Scanner
- `companion.png` — AI Companion
